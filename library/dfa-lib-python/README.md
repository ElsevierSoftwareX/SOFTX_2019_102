Overview
--------

A Python library for Dataflow Analyzer that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install dfa_lib_python


Or clone the repo:

    $ git clone https://gitlab.com/ssvitor/dataflow_analyzer.git
    $ cd library/dfa-lib-python
    $ python setup.py install
